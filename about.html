V1 Module Level Objectives

  1. M1LO6Demonstrate the use of basic HTML tags to create a simple Web page (HTML, this item should be specified once, describing how the file it is in meets this outcome)
  2. M1LO7Demonstrate the use of ordered lists (HTML)
  3. M1LO8Demonstrate the use of external style sheets (HTML, note the external style sheet must exist and contain CSS)
  4. M1LO9Demonstrate the use of inline images (HTML)
  5. M1LO10Demonstrate the use of special characters from extended character sets (HTML, use entities, not cut-and-pasted unicode characters)
  6. M1LO12Create navigation using unordered lists (HTML)
  7. M1LO13Demonstrate using 3 types of hyperlinks: site navigation, targeted location using ids, and external (HTML, this item should be specified 3 times, once on each type of link)
  8. M2LO3Demonstrate style precedence and inheritance in CSS (CSS, this item should be specified 2 times, once where precedence is applied and once where inheritance is applied; see 2-3d Style Specificity and Precedence, and 2-3e Style Inheritance)
  9. M2LO4Demonstrate external, embedded, and inline styles (HTML, this item should be specified 2 times, once each for embedded and inline styles; note external was taken care of with M1LO8; see 2-3b Embedded Style Sheets, and 2-3c Inline Styles)
  10. M2LO5Demonstrate contextual selectors (CSS, specify 2 times, showing two different types of contextual selectors as shown in Figure 2-12, Contextual Selectors, of the textbook)
  11. M2LO6Demonstrate a CSS id and a CSS class selector (CSS, specify 2 times, show a use of id and a use of class, see 2-9b Attribute selectors)
  12. M2LO7Demonstrate a CSS attribute selector other than class or id (CSS, specify 1 time, see 2-9b Attribute selectors)
  13. M2LO11Demonstrate a pseudo-classes and a pseudo-element selector (CSS, specify 2 times, show one pseudo-class and one pseudo-element, see 2-18 Using Pseudo-Classes and Pseudo-Elements)
  14. M2LO12Demonstrate how CSS can change presentation of page elements (CSS, these are styles that impact page elements rather than impacting text on the page)
  15. M2LO14Demonstrate and explain the use of progressive enhancement in CSS (CSS, see 2-6 Employing Progressive Enhancement)
  16. M2LO13Demonstrate how CSS can change presentation of textual elements (CSS, these are styles that impact text that is rendered on the page)
  17. M2LO15Apply colors using CSS (CSS, see 2-5 Working with Color in CSS)

V2 Module Level Objectives

  1. M3LO1Demonstrate the use of a CSS reset style sheet (CSS, note where you have CSS reset rules)
  2. M3LO2Demonstrate floating elements (CSS, note where you have specified a floating element and what its impact is)
  3. M3LO3Demonstrate a clearing floats (CSS, note where you have used a clear and what its impact is)
  4. M3LO6Demonstrate formatting a grid (CSS, note the CSS rules that do this and what their impact is on the HTML; this can be old div-class based grids, or new CSS3 grids)
  5. M3LO8ADemonstrate absolute positioning (CSS, note use in your CSS and describe what its impact is on the HTML)
  6. M3LO8BDemonstrate relative positioning (CSS, note use in your CSS and describe what its impact is on the HTML)
  7. M3LO9Demonstrate fixed, static, or inherited positioning (CSS, note explicit use of one of the three in your CSS and what the impact is on the HTML)
  8. M3LO10Demonstrate an overflow setting (CSS, note explicit use of an overflow setting in your CSS and what the impact is on the HTML)
  9. M3LO11Demonstrate -prefix- style browser extensions use in CSS (CSS note the use in your CSS and what the purpose is)
  10. M4LO1Demonstrate the use of figure boxes (HTML)
  11. M4LO2Demonstrate the use of background images (CSS)
  12. M4LO3Demonstrate the use of rounded corners for borders (CSS)
  13. M4LO4Demonstrate the use of gradients (CSS)
  14. M4LO5Demonstrate the use of text or box shadowing (CSS, note one of these two in your CSS and explain its impact on the HTML)
  15. M4LO6Demonstrate the use of opacity/transparency for images (see 4.2) (CSS)
  16. M4LO7Demonstrate the use of image maps, including determining x and y values (HTML)
  17. M4LO8Demonstrate the use of transformations for 2D or 3D images (CSS, note one of these two in your CSS and explain its impact on the HTML)
